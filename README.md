# CRM Clientes, Empresas e RH

Aplicacao local para cadastrar clientes, empresas, colaboradores, ferias e faltas.
Visual adaptado para a identidade DUOMOLD, com base preta, vermelho e cinza.

## Como abrir

Abra o ficheiro `index.html` no navegador.

## Acesso de demonstracao

O sistema abre direto, sem login. Use o seletor "Ver como" na lateral para alternar entre:

- Admin
- RH
- Funcionario
- Cliente

## O que já inclui

- Entrada direta sem login, ideal para demonstracao.
- Criacao de utilizadores pelo Admin, incluindo email e senha de acesso.
- Criacao de login/senha para clientes no cadastro de clientes.
- Portal do cliente para acompanhar as suas encomendas.
- Pagina de encomendas para o Admin inserir estado, progresso, previsao e atualizacao semanal.
- Paginas funcionais por perfil: Admin, RH, Funcionario e Cliente.
- Painel com totais de clientes, empresas, colaboradores e ferias pendentes.
- Cadastro de clientes com nome, empresa, email, telefone, estado, responsável, cargo e observações.
- Cadastro de empresas com nome, NIF, email, telefone, cidade, setor, morada e observações.
- Cadastro de colaboradores com perfis Admin, RH e Funcionario.
- Gestao de ferias com limite de 30 dias por colaborador.
- Separacao das ferias: 15 dias definidos por Admin/RH e 15 dias pedidos pelo funcionario.
- Validacao de ferias e faltas pelo RH ou Admin.
- Visualizacao por perfil para Admin, RH, Funcionario e Cliente.
- Permissoes por perfil: Admin gere tudo e cria utilizadores; RH valida/inserir ferias e faltas; Funcionario gere os seus pedidos.
- Pesquisa nas tabelas.
- Edição e remoção de registos.
- Exportação dos dados em JSON.
- Dados guardados no próprio navegador.
